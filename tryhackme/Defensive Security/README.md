# Defensive Security Notes

This directory contains personal notes created while completing defensive security-focused TryHackMe rooms.

These notes are **not walkthroughs or room solutions**. Instead, they summarize the key concepts, workflows, and technologies I learned while studying Security Operations, threat detection, incident response, and digital forensics.

The goal is to build a reusable knowledge base that reinforces understanding and can be referenced throughout my cybersecurity journey.

---

# Objectives

- Understand core defensive security concepts
- Build a strong foundation for SOC analyst responsibilities
- Learn how security technologies work together
- Reinforce concepts through structured documentation
- Focus on understanding rather than memorization
